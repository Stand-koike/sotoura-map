# 固定マスタ ピン画像

`config.js` の `FIXED_PIN_ICONS` に対応する PNG をこのフォルダに配置してください。

| ファイル名      | カテゴリ（スプレッドシート J 列） |
|----------------|----------------------------------|
| `toilet.png`   | トイレ                           |
| `parking.png`  | 駐車場                           |
| `viewspot.png` | ビュースポット                   |
| `busstop.png`  | バス停                           |

画像が未配置の場合は絵文字でフォールバック表示されます。
