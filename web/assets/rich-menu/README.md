# リッチメニュー画像仕様（外浦MAP LINE）

**1枚だけ** LINE Official Account Manager に設定（GAS 切替なし）。

## 共通

| 項目 | 値 |
|------|-----|
| レイアウト | 大型 **2×2** |
| サイズ | **2500 × 1686 px** |
| 枚数 | **1枚**（デフォルト ON） |

## ボタン（確定）

| 位置 | ラベル | アクション | LINE に設定する内容 |
|------|--------|------------|---------------------|
| 左上 | ヘルプ | テキスト | `ヘルプ` |
| 右上 | 例文 | テキスト | `例文` |
| 左下 | 登録確認 | テキスト | `登録確認` |
| 右下 | マップを見る | リンク | `https://stand-koike.github.io/sotoura-map/` |

ボタンを押すと GAS が次の2行テンプレを返信します（1行目=タイトル14字、2行目=本文50字）:

```
本日のおすすめ（タイトル14字以内）
（ここに本文）（本文50字以内）
```

文言は `web/gas/line-webhook/02-infrastructure.js` の `buildRichMenuExampleMessage_()` で変更できます。

## 画像

| ファイル | 用途 |
|----------|------|
| `rich-menu-store.png` | **この4ボタン構成用**（LINE にアップロード） |
| `rich-menu-3btn-legacy.png` | 旧3ボタン案（参考・未使用） |
| `rich-menu.pen` | Pencil デザインソース |

## 補足

- **例文** … `例文` と送ると投稿テンプレ（2行）が返る。投稿はそのテキストを編集して手入力
- **初回登録** … 招待コードを手入力。手順は `ヘルプ` または `登録の流れ`（手入力）
- GAS プロパティへのメニュー ID 登録は **不要**
